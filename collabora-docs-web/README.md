# CollaboraDocs Web

#### Vite + TypeScript + React

Package manager: `yarn`
CSS framework: `Tailwind`
Sate Management: `Redux`
Router: `reat-router-dom`

To start app:

```
yarn dev
```

To stop app simply press `q` in terminal.

To generate build:

```
yarn build
```

To preview build locally:

```
yarn preview
```
